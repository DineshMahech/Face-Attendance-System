# Face Recognition Attendance Management System

## 📌 Description
This project is a Face Recognition Based Attendance System developed using Python and OpenCV. It captures face images, trains a model, and marks attendance automatically.

## 🚀 Features
- Face detection and recognition
- Automatic attendance marking
- GUI-based system
- Delete employee feature
- View students

## 🛠️ Technologies Used
- Python
- OpenCV
- Pandas
- Tkinter

## ▶️ How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the project:
python gui.py

## 📂 Project Flow
Register → Capture → Train → Recognize → Attendance

## ⚠️ Note
Training images and attendance files are not included. They will be created automatically.
