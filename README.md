# Face Recognition Attendance Management System

## 📌 Description
This project is a **Face Recognition Based Attendance System** developed using Python and OpenCV.  
It automates the attendance process by detecting and recognizing human faces in real-time.

## 🚀 Features
- ✅ Face Detection using Haar Cascade  
- ✅ Face Recognition using LBPH Algorithm  
- ✅ Automatic Attendance Marking  
- ✅ User-friendly GUI (Tkinter)  
- ✅ Delete Employee Feature  
- ✅ View Registered Students  
- ✅ Sound Notification  

## 🛠️ Technologies Used
- Python
- OpenCV
- Pandas
- Tkinter

## ▶️ How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the project:
python gui.py

## 📂 Project Flow
Register → Capture → Train → Recognize → Attendance

1. Register student and capture images
2. Train model using LBPH algorithm
3. Recognize faces in real-time
4. Store attendance in CSV file
